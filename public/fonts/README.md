# Brand font files

Put licensed Futura PT webfont files here if the organizers provide them.
Do not commit pirated font files. Humanity has enough problems without font licensing drama.

Expected filenames used by `src/styles/main.css`:

- `FuturaPT-Book.woff2` - regular 400
- `FuturaPT-Medium.woff2` - medium 500
- `FuturaPT-Demi.woff2` - bold 700
- `FuturaPT-Heavy.woff2` - heavy 900

If files are missing, the app falls back to Avenir Next / Montserrat / Arial.
